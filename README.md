# Spare Change

## About Spare Change

### Spare Change is an application that allows you to calculate your finances. You input your salary and bills and it will give you a plan for savings, spending, and keep track of your current savings goal.


## How to Run Spare Change
- To run Spare Change you have to open the file in Eclipse.
- Next you will hit the play button while the MainPage.java is openon the top bar to run it.
- The application should run on its own now to use!
